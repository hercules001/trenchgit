# Stream

{% tabs %}
{% tab title="Track Wallet" %}
Steps to add a wallet to your stream

{% stepper %}
{% step %}
#### **Add a new wallet**

On the Tracked Wallets panel (left), click the **“+”** button

<figure><img src="../.gitbook/assets/Screenshot 2026-04-08 at 11.17.44@2x.png" alt="" width="375"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### **Save and start tracking**

Enter a **label, emoji,** and **wallet address** & Click **Add**

<figure><img src="../.gitbook/assets/Screenshot 2026-04-08 at 11.21.22@2x.png" alt="" width="324"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### **Actions**

The wallet will be added to your tracking list. You can then:

* Enable or disable **alerts** for its trades
* Remove the wallet at any time

<figure><img src="../.gitbook/assets/Screenshot 2026-04-08 at 11.24.44@2x.png" alt="" width="375"><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
{% endtab %}

{% tab title="Import Wallets (Bulk)" %}
Steps to add multiple wallets to your stream

{% stepper %}
{% step %}
#### Open Import

Click the **Import** button on the **left panel**

<figure><img src="../.gitbook/assets/Screenshot 2026-04-08 at 11.39.34@2x.png" alt="" width="375"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### Add Wallet Addresses & Confirm

* Add multiple wallet addresses in the **required format**
* Click **Add** to import

<figure><img src="../.gitbook/assets/Screenshot 2026-04-08 at 11.41.55@2x.png" alt="" width="322"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### **Actions**

The wallet will be added to your tracking list. You can then:

* Enable or disable **alerts** for its trades
* **Remove** the wallet at any time
* **Export** your wallet list for backup or reuse

<figure><img src="../.gitbook/assets/Screenshot 2026-04-08 at 11.47.09@2x.png" alt="" width="375"><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
{% endtab %}
