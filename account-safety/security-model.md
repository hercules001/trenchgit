# Security Model

Trench is built on a **non-custodial architecture**, ensuring users retain full control over their funds at all times.

* Wallet-based execution
* No custody of user funds
* Transactions require user action

Trench leverages [**Turnkey**](https://turnkey.com/) for wallet infrastructure, incorporating industry-standard security practices such as:

* Secure key management
* Encrypted operations
* Reliable transaction handling

This ensures a high level of security while maintaining a seamless trading experience.
